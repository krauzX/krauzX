## 👨‍💻 About Me

I am a Software Engineer and Computer Science undergraduate (2nd-year) at **IIIT Kottayam**, specializing in high-performance backend systems, custom language transpilers, and local AI tooling. I operate primarily out of a local MoE pipeline (3B active / RTX 3070) to prototype uncensored, robust architecture.

## 🚀 Featured Engineering & Architecture

### ⚙️ [EmojiScript](https://github.com/krauzX/emoji-script)
> **Custom AST Transpiler & Language Engine**
> Engineered a custom transpiler and multi-mode parser that compiles abstract emoji-based syntax directly into executable JavaScript.
> *Stack: TypeScript, Node.js, Compiler Design*

### 🧠 [Expert Ebook Gen](https://github.com/krauzX/expert-ebook-generator)
> **Multi-Agent Orchestration Engine**
> Built a multi-agent AI manuscript generation system orchestrated dynamically using LangGraph and local Ollama model fallbacks.
> *Stack: LangGraph, Next.js, Ollama, Python*

### 🌌 [Algowiz](https://github.com/krauzX/algowiz)
> **3D Algorithm Rendering Engine**

> *Stack: React Three Fiber, Three.js, Zustand*

> *Stack: Go (Golang), System CLI*

### 🌱 [BluePrint](https://github.com/krauzX/blueprint)
